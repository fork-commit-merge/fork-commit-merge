\section*{Fractions}

The following equation demonstrates fractions:

\[
\frac{a}{b} + \frac{c}{d} = \frac{ad + bc}{bd}
\]

Here:
\begin{itemize}
    \item $a, b, c, d$ are constants
    \item The equation shows how to add two fractions
\end{itemize}

\section*{Integrals}

The following equation represents a definite integral:

\[
\int_a^b f(x)\,dx
\]

Here:
\begin{itemize}
    \item $a$ = lower limit of integration
    \item $b$ = upper limit of integration
    \item $f(x)$ = function being integrated
\end{itemize}

\section*{Matrices}

A 2x2 matrix is written as:

\[
\begin{bmatrix}
a & b \\
c & d
\end{bmatrix}
\]

Here:
\begin{itemize}
    \item $a, b, c, d$ are elements of the matrix
    \item Each value represents a position in rows and columns
\end{itemize}
